Installasi dan Inisialisasi API (Di Komputer Lokal)

  1. Ekstrak file win32.zip, buka command, kemudian ketik cd [LOKASI_EKSTRAK]

  2. Ketik Scripts\activate

  3. Ketik python scale_api\manage.py runserver 0.0.0.0:8000

  4. Apabila tidak terjadi error, buka web browser, buka http://localhost:8000/admin/

  5. Klik link "Scale Configs" untuk mengubah pengaturan

Konfigurasi UI Timbang

  1. Buka web browser, buka http://[IP_SERVER]:8000/

  2. Klik link "Setting"

  3. Klik link "Scale Configs"

Menu

Memulai UI Timbang

  1. Buka web browser, buka http://[IP_SERVER]:8000/, kemudian login

  2. Pilih Role

Menu

Memulai UI Timbang tanpa QC

  1. Isi dan lengkapi form

  2. Klik "Timbang" untuk mengisi otomatis Timbang Masuk

  3. Klik "Submit" untuk menyimpan data Timbang Masuk

  4. Pada saat Timbang Keluar, ketik nomor plat kendaraan, atau klik nomor kendaraan yang ada di tabel bawah

  5. Klik "Timbang" untuk mengisi otomatis Timbang Keluar

  6. Klik "Submit" untuk menyimpan data Timbang Keluar (Contoh Hasil Print)

Menu

Memulai UI Timbang dengan QC

  1. Pada saat QC menyimpan data QC, otomatis akan muncul di tabel bawah, ketik nomor plat kendaraan, atau klik nomor kendaraan yang ada di tabel bawah

  2. Klik "Timbang" untuk mengisi otomatis Timbang Masuk

  3. Klik "Submit" untuk menyimpan data Timbang Masuk

  4. Pada saat Timbang Keluar, ketik nomor plat kendaraan, atau klik nomor kendaraan yang ada di tabel bawah

  5. Klik "Timbang" untuk mengisi otomatis Timbang Keluar

  6. Klik "Submit" untuk menyimpan data Timbang Keluar (Contoh Hasil Print)
